## Overview

A comprehensive desktop Library Lending & Catalog System built with Java Swing and SQLite. The system provides a complete solution for library management with user authentication, role-based access control, book cataloging, borrowing operations, automated fine calculations, hold queues, in-app notifications, and CSV report generation.

## Features

- Secure user authentication with role-based access (Librarian/Member)
- Book catalog management
- Borrowing and returning operations
- Automatic fine calculation
- Hold queue system
- In-app notifications
- CSV report generation

## Quick Start

1. Open the project in NetBeans
1. Add the required libraries to your project (if not added):

- `sqlite-jdbc-3.36.0.3.jar`
- `jbcrypt-0.4.jar`

3. Run the program
4. The database (`library.db`) will be created automatically on first run
